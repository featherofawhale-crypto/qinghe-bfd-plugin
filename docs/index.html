最新版界面

一条时间线，三种复核视角。

主面板负责检测，字幕第二窗口负责快速改字，字体小窗负责 Text+ 样式复核。滚动时界面会随章节切换，就像把一次交付检查拆成几个清晰步骤。

安装方法

打开 DMG，运行一键安装。

当前 macOS 公开包未做 Apple Developer 公证。首次打开时请右键运行安装脚本， 安装完成后重启 DaVinci Resolve。

  1. 1下载 `qinghe-toolbox-v2.0.1-beta.14-macos.dmg`。
  2. 2打开 DMG，右键运行“① 一键安装.command”。
  3. 3重启 DaVinci Resolve。
  4. 4打开“工作区 -> 脚本 -> Edit -> 清何黑帧夹帧检测”。
