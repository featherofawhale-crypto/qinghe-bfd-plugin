最新版界面

主面板、字幕面板、字体面板。

主面板负责扫描时间线风险，字幕面板负责集中校对 SRT 和 Text+，字体面板负责统一 Text+ 字体与样式。 从画面问题到文字细节，按实际交付流程逐段处理。

主面板

交片前一键扫时间线风险。

黑帧、夹帧、重复、透明遮挡、音频、字幕、字体和更新入口集中在同一个控制面板里。检测结果会写回 Resolve 标记，不需要跳出剪辑现场找问题。

核心功能

功能面板的优势，不只是“能检测”。

最大的区别是速度、复杂时间线适应能力和剪辑师工作流。黑帧检测是底座，字幕、Text+、字体和节拍工具是给真实剪辑交付补上的一整套复核能力。

检测速度

长时间线也能快速扫

面向交片前复核优化，不把剪辑师困在等待里。检测完成后直接写回 Resolve 标记，沿着标记修就行。

复杂时间线

多轨、Fusion、OFX 都能进流程

不只看单个素材，复杂模式可按最终画面像素复核，适合包装层、嵌套、Fusion、调色和 OFX 很重的工程。

版本兼容

Resolve 18 / 19 都能用

围绕达芬奇实际 API 限制做兼容处理，字幕、Text+、字体和标记功能尽量覆盖常见 18、19 版本环境。

重复识别

文件改名也不轻易漏

路径重复之外，还用画面指纹辅助判断内容重复。转码、改名、跨文件复用同画面时，也有机会被发现。

字幕面板

查找、替换、删除、撤回

Resolve 18 / 19 可用字幕工具。SRT、字幕轨和 Text+ 集中校对，Text+ 也支持字幕工作流。

SRT 转 Text+

一键把字幕变成包装字

SRT 可转 Text+，适合需要字体、描边、阴影、动效包装的项目；原字幕流程和 Text+ 包装流程能接起来。

字体管理

搜索、收藏、探针、样式库

针对达芬奇 Text+ 的字体管理：中文名、英文名、PostScript 名映射，字体探针，本地保存样式库。

剪辑节拍

BPM 识别，按首拍做标记

不是传统每个鼓点都打的节拍器。可识别 BPM、确定首拍，生成更适合剪辑师卡点的节拍标记。

创新点

把剪辑师真正缺的工具补在一起。

复杂时间线检测

普通检测容易停在素材层，清何把多轨、透明遮挡、Fusion、OFX 和最终像素也纳入复核思路。

Text+ 字幕化

Text+ 不只当标题用，也能接入字幕查找、替换和 SRT 转 Text+，更适合包装字幕交付。

字体管理首创

围绕达芬奇字体痛点做搜索、收藏、字体探针和本地样式库，减少 Text+ 字体替换的反复试错。

样式库更细

保存 Text+ 外观后，可只复制字体，也可复制大部分属性，适合多条字幕统一包装但保留不同文字。

剪辑师节拍

节拍标记按 BPM 和首拍锚点生成，目标是帮助剪辑卡段落，不是把每个鼓点都塞满时间线。

安装方法

打开 DMG，运行一键安装。

当前 macOS 公开包未做 Apple Developer 公证。首次打开时请右键运行安装脚本， 安装完成后重启 DaVinci Resolve。

  1. 1下载 `qinghe-toolbox-v2.0.1-beta.14-macos.dmg`。
  2. 2打开 DMG，右键运行“① 一键安装.command”。
  3. 3重启 DaVinci Resolve。
  4. 4打开“工作区 -> 脚本 -> Edit -> 清何黑帧夹帧检测”。
