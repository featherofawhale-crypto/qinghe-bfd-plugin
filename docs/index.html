最新版界面

单个界面看清楚，再切到下一个。

下载只是入口，真正重要的是它能在 Resolve 里帮你做什么。这里同一时间只展示一张完整截图，自动轮播也可以手动切换，避免界面互相压住看不清。

主面板

交片前一键扫时间线风险。

黑帧、夹帧、重复、透明遮挡、音频、字幕、字体和更新入口集中在同一个控制面板里。检测结果会写回 Resolve 标记，不需要跳出剪辑现场找问题。

核心功能

围绕达芬奇交付质检做完整闭环。

它不是只找“黑一下”的画面，而是把交片前最容易漏掉的画面、文字、字体、音频和标记问题拆成可复查的模块。

画面异常

黑帧、夹帧、卡帧、转场黑场

检测黑帧、可疑黑帧、短夹帧、卡帧、转场黑场和时间线空位，并按风险写入不同颜色标记。

复杂模式

按最终画面像素复核

可渲染 IO 范围后的最终输出画面再分析，用来捕捉调色、OFX、Fusion、遮罩和多轨叠加后才出现的问题。

重复检测

路径重复 + 画面指纹重复

不仅看素材路径，还通过画面指纹识别跨文件内容重复，能发现同一画面被换文件名、转码后再次误用。

遮挡判断

透明、禁用、PNG/PSD 叠加层

识别透明度、禁用片段、半透明素材和 PNG/PSD 遮挡层，减少多轨包装项目里的误报和漏报。

字幕工具

SRT、字幕轨、Text+ 集中校对

优先扫描 SRT 和字幕轨，也能处理 Resolve 文本层与 Text+，搜索高亮不打乱排序，适合边看时间线边改。

字体工具

Text+ 字体扫描、替换和样式库

处理中文字体名、英文名和 Fusion 可用名映射，支持字体预览、批量替换、只复制字体和本地 Text+ 样式库。

音频辅助

单声道检测、BPM 与节拍标记

可检查音频声道异常，识别选中音频 BPM，并按锚点生成节拍标记，方便音乐卡点和后续复核。

标记安全

只清插件标记，保留人工标记

检测结果回写 Resolve 时间线，清理时只处理 `[BFD]`、`[BFD-AUDIO]`、`[QH-BPM]` 等插件标记，避免误删人工备注。

创新点

不是“又一个脚本”，而是为真实剪辑现场补漏洞。

最终像素检测

复杂模式直接看渲染后的实际画面，能发现只在调色、Fusion、OFX、多轨合成后出现的坏帧。

内容指纹重复

重复判断不只依赖文件名和路径，转码、改名、换容器后的同画面重复也有机会被抓出来。

多轨遮挡语义

把 PNG、PSD、透明度、禁用状态和遮挡关系放进判断，面对包装层、角标、字幕条时更贴近最终成片。

字幕第二窗口

字幕面板独立成小窗，适合把 Resolve 时间线留在屏幕主区域，一边播放定位，一边批量改文字。

Resolve 字体映射

针对 Text+ 和 Fusion 的字体名差异做中文名、英文名、PostScript 名映射，减少“电脑有字体但 Resolve 找不到”的情况。

安装方法

打开 DMG，运行一键安装。

当前 macOS 公开包未做 Apple Developer 公证。首次打开时请右键运行安装脚本， 安装完成后重启 DaVinci Resolve。

  1. 1下载 `qinghe-toolbox-v2.0.1-beta.14-macos.dmg`。
  2. 2打开 DMG，右键运行“① 一键安装.command”。
  3. 3重启 DaVinci Resolve。
  4. 4打开“工作区 -> 脚本 -> Edit -> 清何黑帧夹帧检测”。
