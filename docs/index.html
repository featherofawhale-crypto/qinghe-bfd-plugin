安装方法

打开 DMG，运行一键安装

当前 macOS 公开包未做 Apple Developer 公证，首次运行脚本时请右键选择“打开”。 安装完成后重启 DaVinci Resolve。

  1. 1下载 `qinghe-toolbox-v2.0.1-beta.14-macos.dmg`。
  2. 2双击打开 DMG，右键运行“① 一键安装.command”。
  3. 3重启 DaVinci Resolve。
  4. 4打开“工作区 -> 脚本 -> Edit -> 清何黑帧夹帧检测”。
